卷积神经网络手写数字识别

实验思路与实验结果

使用 CNN 进行手写数字的分类，涉及到数据集的导入、图片的预处理、模型的建立与训练、模型的评估

  • 数据集的导入

    直接使用 mnist 提供的 API 导入数据集

  • 图片的预处理

    • 由于均是灰度图，可以将图片转变为单通道以缩小数据规模
    • 对分类变量进行热编码
    • 数据的规范化

  • 模型的建立与训练

    • 卷积层，使用 ReLU 作为激励函数以进行非线性化；核的初始化采用通常使用的he_uniform

    • 池化层，2*2

    • 展平层

    • 全连接层

    • 输出层，使用 SoftMax 作为激励函数

    • 配置随机梯度下降的参数

      learning_rate=0.01, momentum=0.9

    • 编译模型

      遵循通常的配置，即

      loss='categorical_crossentropy', metrics=['accuracy']

    • 训练模型

      代数为5，batch-learning 的大小设为 64

  • 模型的评估

    使用了三种不同的评估角度

    • 准确率与损失

      在程序运行的过程中可以观察到每代的损失和准确率，在经过5代后，准确率为0.9857000112533569。事实上，mnist 数据集上，使用简单的 CNN 都能取得很好的准确率

    • 预测单张图片

      预测下面这张图片的分类结果

      可以看到给出了正确的预测

    • 混淆矩阵

      使用混淆矩阵评估分类的效果。可以看到值均匀集中在对角线上，说明分类的准确率较高
