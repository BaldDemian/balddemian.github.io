动态规划再次入门

Intro

动态规划可以解决哪些问题？

  1. 最值型问题（但是不包括最大值最小化、最小值最大化，这类问题往往通过二分解决）
  2. 可行数量问题（注意动态规划只能求出数量，而无法枚举每种具体做法。枚举具体做法，可能通过回溯解决）

动态问题紧抓：

  1. 状态表示：需要几个维度的状态变量
  2. 状态转移：如何求解状态转移方程？如何将问题分解为若干种已经求解出的子问题？以及如何初始化初始的状态？

背包问题

背包容积有限，如何选物品使得价值最大化？

01 背包问题

物品有限，每种物品至多用 1 次

有 \(N\) 件物品和一个容量是 \(V\) 的背包。每件物品只能使用一次。

第 \(i\) 件物品的体积是 \(v_i\)，价值是 \(w_i\)。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。 输出最大价值。

  • 状态定义

    f[n][v]: 考虑前 n 件物品时，在总体积不超过 v 的前提下，所能获取的最大价值

    显然，f[N][V] 就是题目要求的结果

  • 状态转移

    我们考虑第 \(i\) 件物品选还是不选。选它时，所能获得的最大价值是 \(w_i+f[i-1][v-v_i]\) ；不选它时，所能获取的最大价值是 \(f[i-1][v]\)​，我们取二者的最大值，即构成状态转移方程。这类考虑第 i 件物品选还是不选代表了一类 DP 的状态转移思想

完全背包问题

每种物品可以用无限次

有 \(N\) 种物品和一个容量是 \(V\) 的背包，每种物品都有无限件可用。

第 \(i\) 种物品的体积是 \(v_i\)，价值是 \(w_i\)。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。 输出最大价值。

相比 01 背包问题，完全背包由于每种物品的可用次数无限，因此状态转移的子问题更多一些。具体来讲，对于第 \(i\) 件物品，我们可以选 0 件、1 件、...、\(\lfloor {v/volumns[i]} \rfloor\)​件

计数 DP

一个正整数 \(n\) 可以表示成若干个正整数之和，形如：\(n=n_1+n_2+…+n_k\)，其中 \(n_1≥n_2≥…≥n_k, k≥1\)。

我们将这样的一种表示称为正整数 \(n\) 的一种划分。

现在给定一个正整数 \(n\)，请你求出 \(n\) 共有多少种不同的划分方法。

这道题其实是完全背包问题的一个变体

数位统计 DP（十分痛苦！）

给定两个整数 \(a\) 和 \(b\)，求 \(a\) 和 \(b\) 之间的所有数字中 \(0∼9\) 的出现次数。

例如，\(a=1024，b=1032\)，则 \(a\) 和 \(b\) 之间共有 99 个数如下：

1024 1025 1026 1027 1028 1029 1030 1031 1032

其中 0 出现 1010 次，1 出现 1010 次，2 出现 77 次，3 出现 33 次等等…

解决数位统计问题的关键在于「分情况讨论」

我们尝试实现一个函数 count(n, x)，它用于统计整数 [1, n] 中数位 x 出现的次数。那么，基于前缀和的思想，整数 [a, b] 中数位 x 出现的次数，就是 count(b, x) - count(a - 1, x)

我们再看这个函数如何实现。特别的，我们讨论如何求 count(n, 1)。设 n=abcdefg，我们先考虑如何求出第 4 位为 1 的 [1, n] 中的所有数的数量，即找出形如 xxx1yyy 的数。做如下讨论

  • xxx < abc，那么 xxx 有 abc 种取值。此时 yyy 可以任取，即 1000 种情况。这种情形总共可构造 abc * 1000 种。特别的，当第二个入参为 0 时，这种情况的 xxx 要从 001 开始取，因为 0000efg 其实就是 efg，它的第 4 位其实是不存在的！！！即 (abc-1) * 1000
  • xxx > abc。此时无解
  • xxx = abc。我们接着讨论 d 的取值
    • d < 1，即需要求出使得 abc1yyy <= abc0efg 的可行情况，无解
    • d = 1，即需要求出使得 abc1yyy <= abc1efg 的可行情况，此时 yyy 在[000, efg] 中取值，共 efg + 1 种
    • d > 1，此时 yyy 可任取，共 1000 种情况

由以上讨论，我们可实现 count(n, x) 函数
